# Python Basics Quiz

A comprehensive 50-question quiz covering the fundamentals of Python programming, perfect for reviewing what you learned in the "Basics of Python" module on roadmap.sh.

## Features

- **50 Questions** covering essential Python topics
- **One-by-one answer reveal** - See the correct answer and explanation after each question
- **Immediate feedback** - Know right away if you got it right or wrong
- **Score tracking** - Keep track of your progress throughout the quiz
- **Performance rating** - Get an overall assessment at the end

## Topics Covered

- Variables and Data Types
- Operators (arithmetic, comparison, logical)
- Control Flow (if/else, loops)
- Functions
- Lists, Tuples, Sets, Dictionaries
- String Operations
- Basic Input/Output
- Type Conversion
- Boolean Logic

## How to Use

1. Run the quiz script:
   ```bash
   python3 python_basics_quiz.py
   ```

2. Press Enter to start the quiz

3. For each question:
   - Read the question and options (A, B, C, D)
   - Type your answer (A, B, C, or D)
   - Press Enter to see if you're correct
   - Read the explanation
   - Press Enter to continue to the next question

4. At the end, view your final score and performance rating

## Tips for Success

- Take your time with each question
- Read all options carefully before answering
- Pay attention to the explanations, especially for questions you get wrong
- If you score below 80%, consider retaking the quiz after reviewing the topics
